Embedding PHP in Web Pages

Introduction to Embedding PHP in Webpages

PHP is a server-side scripting language that can be embedded within HTML to create dynamic, interactive web pages. By mixing PHP with HTML, you can generate content that adapts to user input, data from databases, or other server-side information.

Basic PHP Embedding

PHP code is enclosed within <?php ... ?> tags within an HTML file. Here’s a simple example that outputs "Hello, World!" using PHP within an HTML document: 

<!DOCTYPE html>
<html>
<body>

<h1>Welcome to My Website</h1>

<?php
echo "Hello, World!";
?>

</body>
</html>

In this example, the PHP code executes on the server and outputs "Hello, World!" directly into the HTML content.

Embedding Variables in HTML

PHP variables can also be embedded within HTML to make pages more dynamic. Here’s an example that uses a variable to personalize a greeting: 

<!DOCTYPE html>
<html>
<body>

<h1>Personalized Greeting</h1>

<?php
$name = "Jane";
echo "<p>Hello, $name! Welcome to my website.</p>";
?>

</body>
</html>

Here, the PHP variable $name is inserted into the HTML. This code outputs: "Hello, Jane! Welcome to my website."

Using Conditional Statements

PHP allows the use of conditional statements to display different content based on certain conditions. For example: 

<!DOCTYPE html>
<html>
<body>

<h1>Time of Day Greeting</h1>

<?php
$hour = date("H");

if ($hour < 12) {
    echo "<p>Good morning!</p>";
} elseif ($hour < 18) {
    echo "<p>Good afternoon!</p>";
} else {
    echo "<p>Good evening!</p>";
}
?>

</body>
</html>

This script uses PHP’s if statement to display a different greeting based on the current hour.

Embedding Loops

Loops are another powerful feature of PHP that can be embedded within HTML. Here’s an example that generates a list of numbers: 

<!DOCTYPE html>
<html>
<body>

<h1>Number List</h1>

<ul>
<?php
for ($i = 1; $i <= 5; $i++) {
    echo "<li>Number: $i</li>";
}
?>
</ul>

</body>
</html>

This code outputs a numbered list:

  • Number: 1
  • Number: 2
  • Number: 3
  • Number: 4
  • Number: 5

Using PHP to Display Data from Arrays

PHP arrays can also be embedded in HTML to create dynamic lists of items. Here’s an example that displays a list of favorite colors: 

<!DOCTYPE html>
<html>
<body>

<h1>My Favorite Colors</h1>

<ul>
<?php
$colors = array("Red", "Blue", "Green", "Pink");

foreach ($colors as $color) {
    echo "<li>$color</li>";
}
?>
</ul>

</body>
</html>

This code will generate a bulleted list of colors:

  • Red
  • Blue
  • Green
  • Pink

Embedding PHP with Forms

PHP can process form data, allowing users to submit information that PHP can handle on the server. Here’s a basic example that takes a user’s name and displays a personalized message: 

<!DOCTYPE html>
<html>
<body>

<h1>Enter Your Name</h1>

<form method="POST">
    <label>Name: </label>
    <input type="text" name="username">
    <input type="submit" value="Submit">
</form>

<?php
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    $name = htmlspecialchars($_POST["username"]);
    echo "<p>Hello, $name!</p>";
}
?>

</body>
</html>

In this form, after entering a name and submitting, PHP processes the input and outputs a greeting using the submitted name.
