PHP Basics

Introduction to PHP Basics

PHP (Hypertext Preprocessor) is a widely-used open-source scripting language designed for web development. PHP scripts run on the server, making it a server-side language. PHP is known for its ease of integration with HTML and is primarily used to build dynamic and interactive web applications. In this section, we’ll cover the core concepts and syntax fundamentals of PHP, including tokens, variables, constants, data types, operators, control structures, and more.

PHP Tokens

In PHP, tokens are the smallest building blocks in a script. PHP tokens include keywords, identifiers (variable names), constants, strings, operators, and symbols. Here’s a quick overview of common tokens:

  • Keywords: Reserved words that have special meaning, like if, else, echo, etc.
  • Identifiers: Names given to variables, functions, and classes.
  • Constants: Defined values that don’t change during execution.
  • Operators: Symbols used to perform actions, like +, -, *, /.
<?php
    // Example of keywords, identifiers, and operators
    $x = 10; // $x is an identifier, = is an operator, 10 is a constant
    echo $x; // echo is a keyword
?>

Variables in PHP

Variables are used to store data in PHP and are prefixed with a dollar sign ($). Variable names must start with a letter or underscore, followed by letters, numbers, or underscores. PHP is loosely typed, so variables can change types dynamically based on the value assigned. 

<?php
    $name = "John Doe"; // String variable
    $age = 25; // Integer variable
    $is_student = true; // Boolean variable
?>

Variable Scope in PHP

Variable scope defines where a variable can be accessed in a script. PHP has three main scopes:

  • Local Scope: Variables declared within a function and accessible only within that function.
  • Global Scope: Variables declared outside of functions and accessible globally using the global keyword.
  • Static Scope: Variables that retain their value between function calls using the static keyword.
<?php
    $x = 10; // Global variable

    function test() {
        global $x; // Access global variable
        static $y = 0; // Static variable
        $y++;
        echo $x + $y;
    }
?>

Constants in PHP

Constants are like variables but once defined, their values cannot change. Constants are defined using the define() function and are usually written in uppercase letters by convention. 

<?php
    define("SITE_NAME", "My Website");
    echo SITE_NAME; // Outputs: My Website
?>

Data Types in PHP

PHP supports several data types, including:

  • String: A sequence of characters.
  • Integer: Whole numbers.
  • Float: Numbers with decimal points.
  • Boolean: True or false values.
  • Array: A collection of values.
  • Object: An instance of a class.
  • NULL: A variable with no value.

Operators in PHP

PHP provides various operators for performing calculations and operations on variables and values. Common types of operators include:

  • Arithmetic Operators: +, -, *, /, %.
  • Assignment Operators: =, +=, -=, *=, /=, %=.
  • Comparison Operators: ==, !=, >, <, >=, <=.
  • Logical Operators: &&, ||, !.
<?php
    $a = 10;
    $b = 20;
    echo $a + $b; // Outputs: 30
    echo ($a < $b) ? "a is less than b" : "a is greater than or equal to b";
?>

Expressions in PHP

An expression in PHP is any valid combination of variables, constants, operators, and function calls that returns a value. Expressions are often used in control structures and assignments. 

<?php
    $result = (5 * 2) + 3; // Expression that calculates a value
    echo $result; // Outputs: 13
?>

Comments in PHP

Comments are notes for developers and are ignored by the PHP interpreter. PHP supports single-line (// or #) and multi-line (/* ... */) comments. 

<?php
    // This is a single-line comment
    # This is also a single-line comment
    /* 
       This is a multi-line comment
    */
?>

Echo and Print Statements

echo and print are two ways to output text in PHP. echo can take multiple arguments, while print can take only one. 

<?php
    echo "Hello, World!"; // Outputs: Hello, World!
    print "Hello, PHP!"; // Outputs: Hello, PHP!
?>

Control Structures in PHP

Control structures allow you to control the flow of a script. PHP provides branching statements like if-else, switch, and looping statements like while, do-while, and for.

If-Else Statement

<?php
    $num = 10;
    if ($num > 0) {
        echo "Positive";
    } else {
        echo "Negative";
    }
?>

Switch Statement

<?php
    $day = "Monday";
    switch ($day) {
        case "Monday":
            echo "It's Monday!";
            break;
        default:
            echo "Not Monday";
            break;
    }
?>

For Loop

<?php
    for ($i = 1; $i <= 5; $i++) {
        echo "Number: " . $i . "<br>";
    }
?>
